  <h3 align="center">
  
   ## &rightarrow; Sobre o desafio
  </h3>
  
  #### Descrição
  Essa é uma aplicação que usa o Framework Express para demonstra o armazenamento de repositórios de um portfólio. A aplicação permite criar, ler, atualizar e excluir os repositórios e permite que eles recebam "likes". 
  Durante o desenvolvimento da aplicação, a biblioteca **Nodemon** foi utilizada como dependência de desenvolvimento, uma vez que monitora as alterações feitas no código e realiza o reinício automático do projeto em execução.
  
  <h3 align="center">
  
   ## &rightarrow; Iniciando com a aplicação
  </h3>
  
  #### Modelo da aplicação
  Após clonar o modelo da aplicação, execute um dos seguintes comandos na interface de linha de comando para instalar todas as dependências do projeto, conforme o pacote de gerenciamento instalado: **yarn** ou **npm install**. Por padrão, a aplicação será executada utilizando a porta **3333**. Caso deseje altera-lá, acesse o arquivo **app.js** na pasta pricipal do projeto para executar essa ação.
